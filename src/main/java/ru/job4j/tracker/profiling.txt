секунды. Через какое-то время после старта G1 перемещает созданные классы в "G1 Old Gen", "G1 Eden Space",

память упала с 32Мб до 4Мб).
3) При редактировании заявки особых изменений не наблюдается.
4) При добавлении нескольких новых заявок никаких изменений в динамике работы программы не наблюдается. Количество

5) При поиске по id и имени никаких изменений не наблюдается.
6) При удалении заявки никаких изменений не наблюдается.
